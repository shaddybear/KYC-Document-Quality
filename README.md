# KYC Document Quality Assessment

## Overview
A personal project to evaluate and classify KYC documents (PAN, Aadhaar, Passport) for quality and compliance, simulating a real document review workflow.

## Features
- Built a 10-parameter scoring model to check clarity, cropping, glare, expiry, text legibility, and identity match
- Used SQL & Python (Pandas) for data cleaning, duplicate removal, filename validation, and missing/corrupted files detection
- Developed Power BI reports showing rejection frequency, document types, reviewer decisions, and error patterns
- Created a reviewer checklist mimicking real KYC workflow: identity match, address proof, tampering check, signature visibility

## Tools & Skills Used
SQL, Python (Pandas, NumPy), Excel, Power BI, KYC/AML workflow understanding

## Workflow
1. Collected sample KYC documents and data
2. Applied scoring model and validation scripts in Python & SQL
3. Created interactive Power BI reports to track quality metrics
4. Designed reviewer checklist for systematic document evaluation
